# English to Arabic Keyboard Converter

**English to Arabic Keyboard Converter** is a simple Python tool that converts text typed in English (QWERTY) keyboard layout to Arabic letters according to the standard Arabic 101 keyboard layout. It also converts English symbols to their corresponding Arabic letters.

## Features

- Convert English letters to Arabic.
- Convert English symbols like `] [ ; ' , . / \`` to Arabic.
- Supports uppercase and lowercase letters.
- Copy the converted text with one click.
- Clear input and output easily.

## Requirements

- Python 3.x
- Tkinter (usually comes pre-installed with Python)

## Installation

Clone the repository or download the files:

```bash
git clone https://github.com/MohammadSalem-7/converter.git
```


Type your text in English, click **Convert**, and then you can **Copy** or **Clear** the text.
